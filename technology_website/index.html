Information technology is an exact and technical science dealing with information processing, including information processing technologies and technologies for the production of information processing systems.

Internet - a window to the world

The Internet is a worldwide system of connections between computers, also known as a network of networks. In the IT sense, the Internet is the space of IP addresses allocated to hosts and servers connected by network devices, such as network cards, modems, and hubs, communicating via the Internet protocol using a telecommunications infrastructure.
